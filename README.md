## What is RX_CARRENT? ğŸ‡ºğŸ‡¸

- rx_carrent is a script for Retrix.cz server.
- creates three polyzones 1) Cayo - rent, 2) City - rent and 3) City - return
- In Cayo you can rent a boat(Dinghy) or a scooter(Seashark), in town you can rent a Blazer(quad or something like that) or a scooter(Seashark)
- There is a fee for each rental - can be specified in the config
- On the beach, it is possible to return the rented vehicle using ox_target when pointing at the vehicle
- That's all. With love insoomnia
