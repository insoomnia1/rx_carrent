## Co je to RX_CARRENT?

- rx_carrent je script pro server Retrix.cz.
- vytvoří dvě polyzone zóny 1) Cayo, 2) Město - pláž. 
- Na Cayu si můžete půjčit loď(Dinghy) nebo Skůtr(Seashark)
- Za každé půjčení je účtován poplatek
- Na pláži je možné půjčené vozidlo vrátit pomocí ox_targetu když namíříte na vozidlo
- To je vše. S láskou insoomnia
