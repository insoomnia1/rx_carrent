## Co je to RX_CARRENT?

- rx_carrent je script pro server Retrix.cz.
- vytvoří tři polyzone zóny 1) Cayo - půjčení, 2) Město - půjčení a 3) Město - vrácení
- Na Cayu si můžete půjčit loď(Dinghy) nebo Skůtr(Seashark), ve měste zase Blazer(čtyřkolka nebo tak nějak) nebo skůtr(Seashark)
- Za každé půjčení je účtován poplatek - lze určit v configu
- Na pláži je možné půjčené vozidlo vrátit pomocí ox_targetu když namíříte na vozidlo
- To je vše. S láskou insoomnia
